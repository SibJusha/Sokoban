# Sokoban

### Возможности:
* Удобный формат распространения и добавления уровней при помощи Xml
* Таблица лидеров для каждого уровня, где учитывается кол-во шагов и время затраченное на прохождение уровня
* Дополнительные усложнения (ящики и клетки с пометками)

### Использованные технологии:
* .NET
* MonoGame Framework
